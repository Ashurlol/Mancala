This program consists of 3 python files- MancalaBoard, Play and MancalaGUI. Details are in the written report.

Download the folder and run MancalaGUI in src to start the game.
Select player type by enterting their corresponding integers where
RANDOM = 0, MINIMAX = 1, ABPRUNE = 2, HUMAN = 3.

Note: if the input does not work because of version issues, use command: startGame(players(1, player1), players(2, player2)) where player1 and player2 need  to be replaced be the corresponding integers for player type.
